光耦数据分析系统 - 常见问题解决方案
=========================================

问题1: 页面显示 "key...y_right" 或类似的key错误
------------------------------------------------
原因：Streamlit内部生成的组件key冲突，通常是临时性问题

解决方法（按顺序尝试）：
1. 刷新浏览器页面（F5）
2. 强制刷新并清除缓存（Ctrl + F5）
3. 清除浏览器缓存：
   - Chrome: Ctrl + Shift + Delete
   - 选择"缓存的图片和文件"
   - 点击"清除数据"
4. 关闭应用，重新启动
5. 使用浏览器的隐私/无痕模式打开

问题2: 字体显示不正确
----------------------
解决方法：
1. 完全关闭浏览器（所有标签页）
2. 重新启动应用
3. 强制刷新页面（Ctrl + F5）

问题3: 应用启动后无法访问
--------------------------
解决方法：
1. 检查端口8501是否被占用
   - 打开命令提示符
   - 运行: netstat -ano | findstr :8501
   - 如果有进程占用，使用任务管理器结束该进程
2. 使用"停止应用.bat"停止所有Streamlit进程
3. 重新启动应用

问题4: 数据无法加载或显示
--------------------------
解决方法：
1. 确认data文件夹存在且包含数据文件
2. 检查文件格式是否正确（.xlsx, .xls, .csv）
3. 确认文件没有被其他程序占用（如Excel）
4. 查看应用终端窗口的错误信息

问题5: 计算结果不正确
----------------------
解决方法：
1. 检查输入参数是否正确
2. 确认材料折射率设置正确
3. 查看公式说明，确认理解计算逻辑
4. 尝试使用不同的输入组合

问题6: 无法导出数据
--------------------
解决方法：
1. 确认已经完成数据筛选或计算
2. 检查浏览器下载设置
3. 尝试更换浏览器
4. 检查磁盘空间是否充足

问题7: 页面加载缓慢
--------------------
解决方法：
1. 减少同时加载的数据量
2. 关闭不需要的浏览器标签页
3. 清理浏览器缓存
4. 检查系统资源使用情况

联系支持
--------
如果以上方法都无法解决问题，请：
1. 记录错误信息（截图或复制文本）
2. 记录操作步骤
3. 联系技术支持

快捷键参考
----------
- F5: 刷新页面
- Ctrl + F5: 强制刷新（清除缓存）
- Ctrl + Shift + Delete: 打开清除浏览器数据
- Ctrl + Shift + Esc: 打开任务管理器
- Ctrl + C: 停止应用（在终端窗口中）
