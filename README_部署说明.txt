========================================
  Sth-in-EB 部署说明
========================================

【需要复制到目标机器的文件】
1. sth_eb_env.tar.gz  (Python环境包，约204MB)
2. setup_and_run.bat  (启动脚本)
3. app 文件夹         (应用代码)
4. run_app.py         (可选，桌面模式启动)

【使用方法】
双击运行 setup_and_run.bat 即可

首次运行会自动：
- 解压 Python 环境
- 修复环境路径
- 启动 Streamlit 应用

后续运行会直接启动应用

【系统要求】
- Windows 10/11
- 无需安装 Python
- 需要 tar 命令（Windows 10 1803+ 自带）

【注意事项】
- 所有文件需放在同一目录
- 路径中避免使用中文或特殊字符
- 首次启动需要约1-2分钟解压环境
